## 🛠️ Instructions Git (à faire en premier — ~10 min)

1. Clonez le dépôt mis à votre disposition :
   ```bash
   git clone <url_du_repo>
   cd <nom_du_repo>
   ```
2. Créez un dossier à votre nom :
   ```bash
   mkdir NomPrenom
   cd NomPrenom
   ```
3. Copiez les fichiers `THEORIE.md` et `PRATIQUE.md` dans votre dossier et complétez-les
4. Placez vos fichiers PHP (épreuve pratique) dans votre dossier
5. À la fin de l'examen, committez et pushez :
   ```bash
   git add .
   git commit -m "PHP - NomPrenom"
   git push
   ```
